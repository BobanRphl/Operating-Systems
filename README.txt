/**************************************
README for PROJECT 3
**************************************/

The main file (main.c) accepts input file containing process data, output file to print stats and the snapshot interval.

The program is using the following files to run the code for the scheduler algorithm.

'main.c' .

'fifo.c'

'SJF.c'

'STCF.c'

'round_robin.c'

'pre-emptive.c'




/*****To compile the main.c and other files*****/

1. On linux bash navigate to the directory 'proj3'
2. Type make. 
   This will create an executable by the name of make.




/******To run the program*******/

 Type ./make <input_testfile> <output_file> <snapshot_interval>
